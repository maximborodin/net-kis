# Третье задание по ASP.Net Core
## Создание MVC приложения. Часть 2
В данном задании предлагается продолжить написание Outlook--. 
В этой части нужно всего лишь добавить аутентификацию, а именно:

- У пользователя должна быть возможность залогиниться.
- У каждого пользователя отображаются свои заметки.
- Нельзя создавать заметки и встречи не залогинившись.
- При создани встречи нужно иметь возможность добавлять других пользователей.
- При этом у них тоже должна отображаться эта встреча.

Плюсом будет использование всяких красивостей типа bootstrap'a. 
Также не забываем про валидацию и остальные хелперы.
